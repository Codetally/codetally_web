Tally Code Costs Automatically

Codetally helps you determine the costs of developing projects over time. You control the configuration with files in your Github repo. Calculations occur in real-time with every commit.

Tally code costs automatically.

By defining the charges that apply to your repository's commits, you can calculate the cost of developing your project automatically.

Yes, we still support time based cost calculations.

And its pretty painless. Just update a timelog.json file with each commit and we will use that instead for calculating the cost of a commit.
